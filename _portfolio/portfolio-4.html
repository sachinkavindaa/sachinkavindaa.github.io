Aug 2025 – Dec 2025

Associated with University of Nebraska-Lincoln

Developed a bioinformatics workflow for the classification and functional characterization of the CAZy GH5 glycoside hydrolase family, a diverse enzyme family involved in carbohydrate metabolism. The project integrated sequence analysis, domain annotation, network biology, and phylogenetics to identify GH5 subfamilies and explore their functional relationships.

View Project on GitHub

Project Highlights

Protein Dataset Processing

Processed and curated large-scale GH5 protein sequence datasets.

Domain Annotation

Used HMMER and dbCAN to identify conserved carbohydrate-active enzyme domains.

Network Biology

Constructed Sequence Similarity Networks to study protein relationships.

Phylogenetic Analysis

Used MAFFT-based multiple sequence alignment for evolutionary interpretation.

Key Technical Work

  • Processed and curated large-scale protein sequence datasets
  • Performed domain annotation using HMMER and dbCAN
  • Extracted protein modules based on conserved domain architectures
  • Constructed and analyzed Sequence Similarity Networks using SSNpipe
  • Integrated characterized proteins and EC numbers for functional interpretation
  • Visualized biological networks and protein relationships in Cytoscape
  • Conducted multiple sequence alignment and phylogenetic analysis using MAFFT
  • Developed Linux/Bash-based workflows on HPC environments

Interactive Technical Details

View Domain Annotation Step

HMMER and dbCAN were used to identify conserved CAZy-related domains in GH5 protein sequences. These domain annotations helped classify proteins based on conserved functional regions.

View Sequence Similarity Network Step

Sequence Similarity Networks were constructed using SSNpipe to group related GH5 proteins based on sequence similarity. The resulting networks were visualized in Cytoscape to explore subfamily-level clustering.

View Phylogenetic Analysis Step

Multiple sequence alignment was performed using MAFFT, followed by phylogenetic interpretation to understand evolutionary relationships among GH5 subfamilies.

Research Impact

This project supports a better understanding of functional diversity within the CAZy GH5 glycoside hydrolase family. The workflow can help identify enzyme subfamilies, interpret carbohydrate metabolism functions, and support future studies in microbial ecology, enzyme discovery, and computational biology.
